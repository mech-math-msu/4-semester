Задача: Есть \(N\) билетов, студент выучил \(M\) из них. \(A_k\) – студент, стоя \(k\)-м в очереди вытащил хороший билет. \(P(A_k) = ?\)

Доказательство:

Пусть \(H_i\) – до \(k\)-го студента вытащили ровно \(i\) хороших билетов.

\(\Omega = \bigsqcup_i^M H_i\)

\(P(H_i) = \frac{C_M^i\cdot C_{N - M}^{k - 1 - i}}{C_N^{k - 1}}\).

\(P(A_k|H_i) = \frac{M - i}{N - (k + 1)}\) (оставшиеся хорошие поделить на оставшиеся билеты).

\(P(A_k) = \displaystyle \sum_{i = 0}^{M}P(H_i)\cdot P(A_k|H_i) = \frac{C_M^i\cdot C_{N - M}^{k - 1 - i}}{C_N^{k - 1}}\cdot \frac{M - i}{N - (k + 1)} = \frac{M}N\cdot \displaystyle \sum_{i = 0}^{M}P(H_i)\cdot P(A_k|H_i) = \frac{C_{M - 1}^i\cdot C_{N - M}^{k - 1 - i}}{C_{N - 1}^{k - 1}}\), где последняя сумма, также сумма по событиям, составляющим разбиение (просто уменьшили количество билетов на \(1\)), то есть эта сумма равна \(1 \,\,\,\,\blacksquare\)

\[E\left(\frac1{1+\xi}\right) = \displaystyle \sum_{k = 1}^{\infty}\frac1k\cdot P(\{\frac1{1+\xi} = \frac1k\}) = \displaystyle \sum_{k = 1}^{\infty}\frac1k\cdot P(\{\frac1{1+\xi} = \frac1k\})=\] \[= \displaystyle \sum_{k = 1}^{\infty}\frac1k\cdot P(\{\xi = k - 1\}) = \displaystyle \sum_{k = 1}^{\infty}\frac1k\cdot\frac{\lambda^{k- 1}}{(k - 1)!}e^{-\lambda} = \frac1{\lambda}\]

С помощью этой формулы и представления \(\xi = \displaystyle \sum_{i = 1}^{n}\xi_i\) вычислить сумму гипергеометрического распределения.

Доказательство: \[D\left(\displaystyle \sum_{i = 1}^n\xi_i\right) = E\left(\displaystyle \sum_{i = 1}^n\xi_i - E\left(\displaystyle \sum_{i = 1}^n\xi_i\right)\right)^2 = E\left(\displaystyle \sum_{i = 1}^n(\xi_i - E\xi_i)\right)^2=\] \[= \displaystyle \sum_{i = 1}^nD\xi_i + \displaystyle \sum_{i \ne j} E(\xi_i - E\xi_i) E(\xi_j - E\xi_j) = \displaystyle \sum_{i = 1}^nD\xi_i + \displaystyle \sum_{i \ne j} \operatorname{cov}(\xi_i, \xi_j) \,\,\,\,\blacksquare\]

Посчитаем дисперсию гипергеометрического распределения с параметрами \(N, M\) и \(n\). Пусть есть \(N\) шаров, из них \(M\) – белых, остальные – черные. Вынимают \(n\) шаров.

\(\xi\) – число белых шаров среди вынутых.

\(P(\{\xi = m\}) = \frac{C_M^m\cdot C_{N - M}^{n - m}}{C^n_N}, \,\,\,\, m \in \{0,1, \ldots, \min\{M, n\}\}\).

Рассмотрим случайные величины \(\xi_i = 1\) в случае, если достали белый шар и \(\xi_i = 0\) – черный. Тогда

\(\xi_i\) \(0\) \(1\)
\(P\) \(1-\frac{M}N\) \(\frac{M}N\)

\(E\xi = \displaystyle \sum_{i = 1}^{n}\xi_i = n\cdot \frac{M}N\)

\(D\xi_i = E(\xi_i - E\xi_i)^2 = E\left(\xi_i - \frac{M}N\right)^2 = \frac{M}N - \left(\frac{M}N\right)^2\)

\(\operatorname{cov}(\xi_i, \xi_j) = E\xi_i\xi_j - E\xi_iE\xi_j = \frac{M(M - 1)}{N(N - 1)} - \left(\frac{M}N\right)^2\)

\[D\xi = D\left(\displaystyle \sum_{i = 1}^{n}\xi_i\right)=\displaystyle \sum_{i = 1}^nD\xi_i + \displaystyle \sum_{i \ne j} \operatorname{cov}(\xi_i, \xi_j) =\] \[= n \cdot \left(\frac{M}N - \left(\frac{M}N\right)^2\right) + n(n - 1)\left(\frac{M(M - 1)}{N(N - 1)} - \left(\frac{M}N\right)^2\right)=\] \[=n\frac{M}N\left(1 - \frac{M}N\right) + n(n - 1)\frac{M}N\left(\frac{M - 1}{N - 1} - \frac{M}N\right)=\] \[=n\frac{M}N\left(1 - \frac{M}N\right) + n(n - 1)\frac{M}N\left(\frac{M - N}{(N - 1)N}\right)=\]
