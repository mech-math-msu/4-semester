Лекция 5

Распределение сумм независимых случайных величин

Неравенства Чебышёва и Маркова

Утверждение: (первое неравенство Чебышёва) Если случайная величина \(\xi\) неотрицательна, то \(\forall c > 0 \,\,\,\, P(\{\xi \geqslant c\}) \leqslant \frac1c\cdot E\xi\).

Доказательство: см. действан \(\,\,\,\,\blacksquare\)

Утверждение: (второе неравенство Чебышёва) Если случайная величина \(\xi\) имеет конечную дисперсию, то \(\forall c > 0 \,\,\,\, P(\{|\xi - E\xi| \geqslant c\}) \leqslant \frac1{c^2}\cdot \operatorname{D}\xi\).

Доказательство: \(\,\,\,\,\blacksquare\)

Следствие: (Неравенство Маркова) \(\forall r \in \mathbb{R} \,\,\,\,\forall c > 0 \,\,\,\, P(\{|\xi| \geqslant c\}) \leqslant \frac1{c^r}\cdot E|\xi|^r\).

Доказательство:

Замечание: Второе неравенство Чебышёва в употребимой форме: \(P(\{|\xi - E\xi| \geqslant c\cdot \sqrt{\operatorname{D}\xi}\}) \leqslant \frac1{c^2}\).

Закон больших чисел

Теорема: (закон больших чисел в форме Чебышёва) Пусть \(\xi_1, \ldots, \xi_N\) попарно независимы (достаточно некорреллированности) и \(\forall i \,\,\,\, \operatorname{D}\xi_i < C, \,\,\,\, C > 0\). Обозначим \(\xi = \displaystyle \sum_{i = 1}^{N}\xi_i\) Тогда \(\forall c > 0:\)

\[\displaystyle \lim_{n\to \infty}P\left(\left\{\left|\frac{\xi}{n} - E\frac{\xi}n\right| \geqslant c \right\}\right) \to 0\]

Доказательство:

По второму неравенству Чебышёва \(P\left(\left\{\left|\frac{\xi}{n} - E\frac{\xi}n\right| \geqslant c \right\}\right) \leqslant \frac{\operatorname{D}\xi}{n^2c^2} =\) в силу независимости \(= \frac1{n^2c^2} \displaystyle \sum_{i = 1}^{N}\operatorname{D}\xi_i \leqslant \frac{NC}{n^2c^2} \,\,\,\,\blacksquare\)

Распределения сумм Бернуллиевских случайных величин

Теорема: Пусть \(\nu_p\) – число успехов в серии из \(n\) независимых испытаний с вероятностью успеха в каждом конкретном случае \(p\) (испытание Бернулли). Тогда \(\forall \varepsilon > 0\)

\[\displaystyle \lim_{n\to \infty}P\left(\left\{\left|\frac{\nu_p}{n} - p\right| \geqslant \varepsilon \right\}\right) \to 0\]

Доказательство: следует из закона больших чисел \(\,\,\,\,\blacksquare\)

Предельная теорема Пуассона
