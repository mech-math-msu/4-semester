Лекция 7

Общее определение вероятностного пространства

Определение: Пусть \((\Omega, \mathcal{F})\) – сигма-алгебра с единицей \(\Omega\).

Отображение \(P: \mathcal{F} \to \mathbb{R}\) – вероятностная мера, если

  1. (неотрицательность) \(\forall A \in \mathcal{F} \,\,\,\, P(A) \geqslant 0\),
  2. (нормированность) \(P(\Omega) = 1\),
  3. (счетная аддитивность) \(\forall A_1, \ldots, A_n, \ldots: \,\,\,\, A_i \cap A_j = \varnothing, \,\,\,\, i \ne j \,\,\,\, P\left(\displaystyle \bigsqcup_{i = 1}^{\infty}A_i\right)P\left(\displaystyle \sum_{i = 1}^{\infty}A_i\right) = \displaystyle \sum_{i = 1}^{\infty}P(A_i)\).

Определение: Тройка \((\Omega, \mathcal{F}, P)\) – вероятностное пространство.

см. семинары

Распределение случайной величины

Определение: Пусть \(\xi\) – случайная величина на пространстве \((\Omega, \mathcal{F}, P)\). Распределение случайной величины \(\xi\) – это функция \(F_{\xi}(x) = P\{w \in \Omega: \,\,\,\, \xi(w) \leqslant x\}\).

Лемма: Вероятность попадания случайной величины в интервал \((a, b)\) – это \(F_{\xi}(b) - F_{\xi}(a)\).

Доказательство: \(\,\,\,\,\blacksquare\)

Свойства функции распределения

Утверждение: Пусть \(\xi\) – случайная величина на пространстве \((\Omega, \mathcal{F}, P)\). Тогда для \(F_{\xi}\) выполнено:

  1. \(\forall x \in \mathbb{R} \,\,\,\, 0 \leqslant F_{\xi}(x) \leqslant 1\)
  2. \(F_{\xi}\) неубывающая функция,
  3. \(F_{\xi}(-\infty):= \displaystyle \lim_{x\to -\infty}F_{\xi}(x) = 0\) и \(F_{\xi}(+\infty):= \displaystyle \lim_{x\to +\infty}F_{\xi}(x) = 1\),
  4. \(F_{\xi}\) непрерывна справа в любой точке.

Доказательство: \(\,\,\,\,\blacksquare\)

Определение: Случайная величина \(\xi\) непрерывна, если непрерывна ее функция распределния.

Утверждение: (критерий непрерывности) \(\xi\) непрерывна \(\iff P\{\xi = x_0\} = 0 \,\,\,\, \forall x_0 \in \mathbb{R}\).

Доказательство:

\(\xi\) непрерывна \(\implies \forall \varepsilon > 0 \,\,\,\, \exists \delta > 0: \,\,\,\, \forall x (|x - x_0| < \delta) \,\,\,\, |F_{\xi}(x) - F_{\xi}(x_0)| = |P\{w: \,\,\,\,x_0 \leqslant \xi(w) \leqslant x\}| < \varepsilon \,\,\,\,\blacksquare\)

Абсолютно непрерывные случайные величины

Определение: Случайная величина \(\xi\) – абсолютно непрерывна, если существует функция \(f: \mathbb{R} \to \mathbb{R}\), такая, что

  1. \(\forall x \in \mathbb{R} \,\,\,\, f(x) \geqslant 0\),
  2. \(\displaystyle\int\limits_{-\infty}^{+\infty} f(x)dx = 1\),
  3. \(\forall x \in \mathbb{R} \,\,\,\, F_{\xi}(x) = \displaystyle\int\limits_{-\infty}^x f(t)dt\).

В таком случае \(f(x)\) – плотность распределения случайной величины \(\xi\).

Лемма: \(\displaystyle\int\limits_a^b f_{\xi}(x)dx = F_{\xi}(b) - F_{\xi}(a)\).

Примеры плотностей абсолютно непрерывных случайных величин

Плотность равномерного распределения на отрезке \([a, b]\)

\(f(x) = \frac1{b - a}, \,\,\,\, x \in [a, b], \,\,\,\, x = 0, \,\,\,\, x\not\in [a, b]\)

Плотность экмпоненциального распределения с параметром \(\lambda\)

\[f(x) = \lambda\cdot e^{- \lambda x}, \,\,\,\, x \geqslant 0, \,\,\,\, 0, \,\,\,\, x < 0\]

Плотность распределения Коши (с параметром сдвига равным нулю и масштаба равным единице)

\[f(x) = \frac1{\pi(1 + x^2)}\]

Плотность распределения Парето с параметром \(a > 0\)

\[f(x) = \frac{a}{x^{a+ 1}}, \,\,\,\, x \geqslant 1\]

Плотность гауссовского (нормального) распределения \(\mathcal{N}(\mu, \sigma), \,\,\,\, \sigma > 0\)

\[f(x) = \frac1{\sqrt{2\pi}\sigma}e^{-\frac{(x - \mu)^2}{2\sigma^2}}\]

Усиленный закон больших чисел

Теорема: (Усиленный закон больших чисел для схемы Бернулли) Пусть \(\theta_1, \ldots, \theta_n\) – независимые бернуллиевские случайные величины, принимающие значения \(1\) и \(0\) с вероятностями \(p\) и \(q = 1 - p\). Обозначим \(\nu_n = \displaystyle \sum_{i = 1}^{n}\theta_i\). Тогда \(P\left\{\displaystyle \lim_{n\to \infty}\left|\frac{\nu_n}{n} - p\right| = 0\right\} = 1\) (при большом числе экспериментов вероятность средняя вероятность успешных событий совпадает с вероятностью успеха)
