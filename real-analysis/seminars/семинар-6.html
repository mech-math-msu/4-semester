Неметризуемость поточечной сходимости

Утверждение: Пусть \(\{f_{n,k}\}\) – элементы метрического пространства и \((1)\forall n \,\,\,\, f_{n,k} \to f_n, \,\,\,\, k \to \infty\), и \((2)f_n \to f, \,\,\,\, n \to \infty\). Тогда \(\exists \{k_n\}: \,\,\,\, f_{n, k_n} \to f\).

Доказательство:

В силу \((1) \,\,\,\,\forall n \,\,\,\, \exists k_n: \,\,\,\, \rho(f_{n, k_n}, f_n) < \frac1n\)

\[\rho(f_{n, k_n}, f) < \rho(f_{k_n}, f_n) + \rho(f_n, f) < \frac1n + \rho(f_n, f) \to 0, \,\,\,\, n \to \infty \,\,\,\,\blacksquare\]

Определение: Нулевой класс Бэра – это непрерывные функции,

\(n\)-й класс Бэра – это такие \(f\), не лежащие в предыдущих классах Бэра, что существует последовательность функций из \(n - 1\) класса Бэра, сходящаяся к \(f\).

Утверждение: \(\exists f_{n,k} \in C([0, 1]): \,\,\,\, \forall X \,\,\,\, \displaystyle \lim_{n\to \infty}\underbrace{\displaystyle \lim_{k\to \infty}}_{f_n(x)}f_{n, k}(x) = d(x)\), где \(d(x)\) – функция Дирихле.

Доказательство:

Пусть \(r_1, \ldots, r_n\) – рациональные точки отрезка \([0, 1]\).

Пусть \(f_n(x) = \begin{cases} 1, \,\,\,\, x\in \{r_1, \ldots, r_n\}\\ 0, \,\,\,\, x\not\in \{r_1, \ldots, r_n\} \end{cases}\)

Последовательность \(f_{n, k}(x)\) – это сжимающиеся к \(r_1, \ldots, r_n\) треугольники.

Следствие: (из задачи) Поточечная сходимость функций не задается метрикой.

Доказательство: \(\,\,\,\,\blacksquare\)

Пример неограниченной функции на \([0, 1]\) интегрируемой по Лебегу

Неравенство Чебышёва

Теорема: (Неравенство Чебышёва) Если \(f\) неотрицательная интегрируемая, то \(\forall c > 0 \,\,\,\, \mu\{x: \,\,\,\, f(x) \geqslant c\} \leqslant \frac1c \displaystyle\int\limits_X f(x)d\mu\)

\(f\) может быть неограничена, но не слишком.

Доказательство: Рассмотрим функцию \(h(x) = c\cdot \chi_{\{x: \,\,\,\, f(x) \geqslant c\}} \in J(f) \implies \displaystyle\int\limits_X h(x)d\mu \leqslant \displaystyle\int\limits_X f(x)d\mu \,\,\,\,\blacksquare\)

Следствие: Пусть \(f \geqslant 0\) и \(\displaystyle\int\limits_X f(x)d\mu = 0\). Тогда \(f = 0\) почти всюду.

Доказательство: Хотим \(f = 0\) почти всюду, то есть в силу неотрицательности \(f\) хотим \(\mu\{x: \,\,\,\, f(x) > 0\} = 0\).

\(\{x: \,\,\,\, f(x) > 0\} = \displaystyle\bigcup_{n = 1}^{\infty}\{x: \,\,\,\, f(x) > \frac1n\}\), по неравенству Чебышёва \(\mu\{x: \,\,\,\, f(x) > \frac1n\} \leqslant n\cdot \displaystyle\int\limits_X f(x)d\mu = 0 \,\,\,\,\blacksquare\)

Теорема: Если \(f\) измерима и \(g\) интегрируема по Лебегу, \(|f| \leqslant g\), то \(f\) интегрируема по Лебегу.

Доказательство:

Следствие: Пусть \(f \geqslant 0\) измерима, \(E_k = \{x: \,\,\,\, k - 1 \leqslant f(x) \leqslant k\}\), \(\mu X < \infty\).

Тогда \(f \in \mathcal{L}(X) \iff \displaystyle \sum_{k = 1}^{\infty}k\mu E_k < \infty\).

Доказательство: \(\displaystyle \sum_{k = 1}^{\infty}k\mu E_k = \displaystyle\int\limits_X g(x)d\mu = \displaystyle \sum_{k = 1}^{\infty}k\chi_{E_k}(x)\).

По построению на каждом \(E_k \,\,\,\, f(x) \leqslant g(x)\) и дополнив нулями, где надо имеем \(X = \displaystyle\bigsqcup_{k = 1}^{\infty}E_k\).

Ряд сходится \(\implies f(x) \in \mathcal{L}(X)\) по утверждению \(\implies g(x) \leqslant f(x) + 1\), а \(1\) измерима, значит по предыдущей теореме \(g(x) \in \mathcal{L}(X) \,\,\,\,\blacksquare\)

Пример на использование предыдущего следствия

Утверждение: Пусть \(f(x) = \frac1x\) на \((0, 1]\) и \(f(0) = 0\). Тогда \(f \not\in \mathcal{L}([0, 1])\).

Доказательство: \(E_k = \{x: \,\,\,\, k - 1 \leqslant f(x) \leqslant k\} = \left(\frac1k, \frac1{k - 1}\right]\).

\(\displaystyle \sum_{k = 1}^{\infty}k\mu E_k = \infty \,\,\,\,\blacksquare\)

Задача: \((5)\) Построить \(f\) и \(A_n: \,\,\,\, [0, 1] = \displaystyle\bigsqcup_{n = 0}^{\infty}A_n \,\,\,\, \forall n \,\,\,\, f \in \mathcal{L}(A_n) \,\,\,\, \displaystyle \sum_{n = 0}^{\infty}\left|\displaystyle\int\limits_{A_n} f(x)d\mu\right| < \infty\), но \(f\not\in \mathcal{L}([0, 1])\).

Доказательство: Пусть \(A_n = \left(\frac1{n+1}, \frac1n\right]\) и пусть \(A_0 = \{0\}\).

Середина каждого \(A_n\) – это \(\frac12\left(\frac1{n+1}+\frac1n\right) = \frac{2n+1}{2n(2n+1)}\).

Мера \(\mu(\frac1{n+1}, \frac{2n+1}{2n(2n+1)}] = \frac1{2n(n + 1)}\)

На каждом \(A_n\) определелим \(f_n = \frac{-1}{\frac1{2n(n + 1)}}\cdot \chi_{(\frac1{n+1}, \frac{2n+1}{2n(2n+1)}]}(x) + \frac{1}{\frac1{2n(n + 1)}}\cdot \chi_{(\frac{2n+1}{2n(2n+1)}, \frac1{n}]}(x)\)

Пусть \(f = f_n, \,\,\,\, x \in A_n\)

Получили, что на каждом \(A_n\) \(f\) интегрируема и интеграл равен нулю, а на \([0, 1] \,\,\,\, f\) не интегрируема \(\,\,\,\,\blacksquare\)
