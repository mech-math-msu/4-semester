Лекция 3

Продолжение меры по Лебегу

Теорема: Пусть задана сигма-аддитивная мера \(m\) на полукольце \(S\) с единицей \(X\). Система \(\mathfrak{M}\) измеримых множеств является алгеброй с единицей \(X\).

Доказательство:

Пусть \(A, B \in \mathfrak{M}\).

\(X\setminus A\) измеримо, так как, в силу измеримости \(A\) имеем \(\forall \varepsilon > 0 \,\,\,\, \exists A_{\varepsilon}: \,\,\,\, \mu^{*}(A\triangle A_{\varepsilon}) < \varepsilon\), а также \((X\setminus A)\cup (X\setminus A_{\varepsilon}) = A\triangle A_{\varepsilon}\).

\(A\cup B\) измеримо, так как можем выбрать \(A_{\varepsilon}\) и \(B_{\varepsilon}\), такие, что \(\mu^{*}(A\triangle A_{\varepsilon}) < \frac{\varepsilon}2\) и \(\mu^{*}(B\triangle B_{\varepsilon}) < \frac{\varepsilon}2\). Имеем: \((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon}) \subset (A\triangle A_{\varepsilon})\cup (B\triangle B_{\varepsilon})\). Далее из суббаддитивности меры Лебега получаем \(\mu^{*}((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon})) \leqslant \mu^{*}((A\triangle A_{\varepsilon})\cup (B\triangle B_{\varepsilon})) \leqslant \varepsilon\)

\(A\cap B\) измеримо, так как \(A\cap B = X \setminus ((X\setminus A)\cup (X \setminus B))\)

\(A\triangle B\) измеримо, так как \(A\triangle B = (A\setminus B)\cup (B \setminus A) \,\,\,\,\blacksquare\)

Теорема: Пусть задана сигма-аддитивная мера \(m\) на полукольце \(S\) с единицей \(X\). Тогда мера Лебега является корректно определенной мерой на \(\mathfrak{M}\).

Доказательство:

Пусть \(A, B \in \mathfrak{M}\) и \(A\cap B = \varnothing\).

  1. Выберем \(A_{\varepsilon}\) и \(B_{\varepsilon}\), такие, что \(\mu^{*}(A\triangle A_{\varepsilon}) < \frac{\varepsilon}2\) и \(\mu^{*}(B\triangle B_{\varepsilon}) < \frac{\varepsilon}2\).

  2. \((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon}) \subset (A\triangle A_{\varepsilon})\cup (B\triangle B_{\varepsilon})\). Из суббаддитивности меры Лебега получаем \(\mu^{*}((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon})) \leqslant \mu^{*}((A\triangle A_{\varepsilon})\cup (B\triangle B_{\varepsilon})) \leqslant \varepsilon\).

  3. \(A_{\varepsilon}\cup B_{\varepsilon} \subset (A \cup B)\cup ((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon})) \implies \mu^{*}(A_{\varepsilon}\cup B_{\varepsilon}) \leqslant \mu^{*}(A \cup B) + \mu^{*}((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon}))\)

  4. \(A \subset A_{\varepsilon} \cup (A\triangle A_{\varepsilon}) \implies \mu^{*}(A) \leqslant \mu^{*}(A_{\varepsilon}) + \mu^{*}(A\triangle A_{\varepsilon}) < \mu^{*}(A_{\varepsilon}) + \frac{\varepsilon}2 \implies m(A_{\varepsilon}) = \mu^{*}(A_{\varepsilon}) \geqslant \mu^{*}(A) - \frac{\varepsilon}2\) и \(m(B_{\varepsilon}) = \mu^{*}(B_{\varepsilon}) \geqslant \mu^{*}(B) - \frac{\varepsilon}2\)

  5. \(A\cap B = \varnothing \implies (A_{\varepsilon}\cap B_{\varepsilon}) \subset (A\triangle A_{\varepsilon})\cup (B\triangle B_{\varepsilon}) \implies m(A_{\varepsilon} \cap B_{\varepsilon}) = \mu^{*}(A_{\varepsilon} \cap B_{\varepsilon}) \leqslant \mu^{*}(A\triangle A_{\varepsilon})+ \mu^{*}(B\triangle B_{\varepsilon}) < \varepsilon\)

Имеем из 3 и 4:

  1. \(\mu^{*}(A_{\varepsilon}\cup B_{\varepsilon}) = \mu^{*}(A_{\varepsilon})+ \mu^{*}(B_{\varepsilon}) - \mu^{*}(A_{\varepsilon}\cap B_{\varepsilon}) \geqslant \mu^{*}(A) + \mu^{*}(B) - 2 \varepsilon\)

Имеем из 1, 2 и 5:

\(\mu^{*}(A \cup B) \geqslant \mu^{*}(A_{\varepsilon}\cup B_{\varepsilon}) - \mu^{*}((A\cup B)\triangle (A_{\varepsilon}\cup B_{\varepsilon})) \geqslant \mu^{*}(A) + \mu^{*}(B) - 3 \varepsilon\)

В силу произвольности \(\varepsilon\): \(\mu^{*}(A \cup B) \geqslant \mu^{*}(A) + \mu^{*}(B)\), а также в силу субаддитивности \(\mu^{*}(A \cup B) \leqslant \mu^{*}(A) + \mu^{*}(B) \,\,\,\,\blacksquare\)

Теорема: Пусть задана сигма-аддитивная мера \(m\) на полукольце \(S\) с единицей \(X\). Тогда система измеримых множеств \(\mathfrak{M}\) является \(\sigma\)-алгеброй.

Доказательство:

Пусть \(A = \cup_{k = 1}^{\infty}A_k\). Достаточно доказать для непересекающихся \(A_k\), так как иначе воспользуемся леммой.

Из аддитивности меры имеем: \(\displaystyle \sum_{k = 1}^{N}\mu A_k = \mu^{*}\displaystyle \bigsqcup_{k = 1}^{N}A_k \leqslant \mu^{*}A\), а значит ряд сходится.

Имеем \(\forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\, D = \mu^{*}\left(\bigsqcup_{N = 1}^{\infty}A_k\right) \leqslant^{(1)} \displaystyle \sum_{k = N}^{\infty}\mu A_k < \frac{\varepsilon}2\). \((1)\) из субаддитивности меры Лебега.

Из аддитивности: \(C = \displaystyle \sum_{k = 1}^{N}\mu A_k\) измеримо, а значит \(\exists C_{\varepsilon}: \,\,\,\, \mu^{*}(C\triangle C_{\varepsilon}) < \varepsilon\).

\(A\triangle C_{\varepsilon} = (C \sqcup D)\triangle C_{\varepsilon} = (C\triangle C_{\varepsilon})\sqcup (D\triangle C_{\varepsilon}) \subset (C\triangle C_{\varepsilon})\sqcup D \,\,\,\,\blacksquare\)

Теорема: Пусть задана сигма-аддитивная мера \(m\) на полукольце \(S\) с единицей \(X\). Тогда мера Лебега сигма-аддитивна на сигма-алгебре измеримых множеств \(\mathfrak{M}\).

Доказательство: Согласно теореме, мера Лебега сигма-аддитивна, если для любых \(A\) и \(A_k\) из \(\mathfrak{M}\), удовлетворяющих условию \(A = \displaystyle\bigsqcup_{k = 1}^{\infty}\), выполнено неравенство \(\mu A \leqslant \displaystyle \sum_{k = 1}^{\infty}\mu A_k\). А неравенство выполнено, так как мера Лебега субаддитивна \(\,\,\,\,\blacksquare\)

Определение: Классическая мера Лебега на отрезке – это продолжение по Лебегу стандартной меры на полукольце промежутков \(m[a, b] = \ldots = m(a, b) = b - a\).

\(\sigma\)-конечные меры

Определение: Пусть \(m\) – \(\sigma\)-аддитивная мера на полукольце \(S\) с единицей \(X\). \(m\) – сигма-конечна, если \(\exists\{X_k\} \,\,\,\, X_k \in S: \,\,\,\, mX_k < \infty,\,\,\,\, X = \bigsqcup_{k = 1}^{\infty}X_k, \,\,\,\, mX = +\infty\).

Пример: \(\mathbb{R} = \bigsqcup_{n = \mathbb{Z}}[n, n + 1)\), \(m\) – стандартная мера.

Определение: (ограничение полукольца на элемент) Пусть \(S\) – система множеств. \(A \in S\). Тогда \(S\cap A := S_{|A} := \{B \in S|\,\,\,\, B \subset A\} = \{B\cap A|\,\,\,\, B \in S\}\).

Лемма: Если \(S\) полукольцо(кольцо, сигма-кольцо), \(A \in S\). Тогда \(S_{|A}\) – полукольцо (кольцо, сигма-алгебра) с единицей \(A\).

Доказательство:

Если \(B_1, B_2 \in S, \,\,\,\, B_1 \subset A, \,\,\,\, B_2 \subset A\), то \(B_1 \cap B_2 \subset A\) и \(B_1 \cap B_2 \in S\) и дополняющие множества тоже вложены \(\,\,\,\,\blacksquare\)

Измеримость множества по Лебегу в случае сигма-конечной меры

Определение: Пусть \(m\) – сигма-конечная мера на полукольце \(S\) с единицей \(X\), \(X = \bigsqcup_{n = 1}^{\infty}X_n, \,\,\,\, X_n \in S, \,\,\,\, m X_n < +\infty\).

Множество \(A \subset X\) – измеримо по Лебегу, если \(\forall n \,\,\,\,A\cap X_n\) измеримо по Лебегу относительно сужения \(\mu_n\) меры \(m\) на \(S_{|X_n}\).

Мера Лебега измеримого множества \(A\) – это \(\mu A = \displaystyle \sum_{n = 1}^{\infty}\mu_n (A\cap X_n)\).

Теорема:
  1. Определение измеримого по Лебегу множества корректно в случае сигма-конечной меры, то есть не зависит от \(\{X_n\}\)
  2. \(\mathfrak{M}\) – \(\sigma\)-алгебра, а \(\mu\) – сигма-аддитивная мера на ней.

Доказательство:

без доказательства \(\,\,\,\,\blacksquare\)

Теорема: (структура измеримых по Лебегу множеств) Пусть \(\mu\) продолжение по Лебегу конечной или сигма-конечной меры \(m\), \(\mathfrak{M}\) система измеримых множеств. \(\forall A \in \mathfrak{M}: \,\,\,\, \mu A < +\infty \,\,\,\, \exists B^k_n \in R(S)\), такие, что:

  1. \(B^k_n \subset B^{k +1}_n \,\,\,\, \forall n, k \in \mathbb{N}\)
  2. Пусть \(B_n = \displaystyle\bigcup_{k = 1}^{\infty}B^k_n\). Тогда \(B_n \supset B_{n + 1} \forall n\)
  3. Пусть \(B = \displaystyle\bigcap_{n = 1}^{\infty}B_n\). Тогда \(B \supset A \forall n\) и \(\mu(B\setminus A) = 0\).

Доказательство:

\(\forall n \,\,\,\, \exists C_n = \displaystyle\bigcup_k \Delta^k_n \supset A: \,\,\,\, \mu(C_n) \leqslant \displaystyle \sum_{k}^{\infty}\mu(\Delta^k_n) < \mu(A) + \frac1n\).

\(B_n = \displaystyle\bigcap_{k = 1}C_k\). \(\exists D^k_n \in S: \,\,\,\, B_n = \displaystyle\bigcup_{k}D^k_n\). Пусть \(B^k_n = \displaystyle\bigcup_{i = 1}^kD^i_n \,\,\,\,\blacksquare\)

Борелевская алгебра

Определение: Наименьшая \(\sigma\)-алгебра на топологическом пространстве, содержащая все открытые множества – борелевская алгебра. Обозначается: \(\mathcal{B}\).

Лемма: Если \(\{M_{\alpha}\}\) – семейство сигма-алгебр каждая с единицей \(X\), то \(\cap_{\alpha}M_{\alpha} = M\) – сигма-алгебра с единицей \(X\).

Доказательство: \(\,\,\,\,\blacksquare\)

Замечание: Пересекая все сигма-алгебры, содержащие открытые множества получим борелевскую.
