Лекция 10

Поверхностные интегралы

Определение: Рассмотрим гладкую невырожденную поверхность \(S\) в \(\mathbf{R}^3\) и пусть у \(S\) нет кратных точек (самопересечений), то есть параметризация \(\mathbf{\varphi}\) – биекция. Граница \(\partial S\) поверхности \(S\) – это \(\mathbf{\varphi}(\partial \Omega)\). (\(\partial S\) – это простая замкнутая кривая в \(\mathbf{R}^3\))

Определение: Пусть все обозначения взяты из определения только \(n = 3\) и пусть \(f(x, y, z) \in C(S)\). Если \(\exists \displaystyle \lim_{d\to 0} \displaystyle \sum_{i, j = 1}^{N}f(x(u_i, v_j), y(u_i, v_j), z(u_i, v_j))\mu(\Pi_{ij}) = I\), то \(I\) – поверхностный интеграл первого рода по поверхности \(S\). Обозначается: \(\displaystyle\iint\limits_S f(x, y, z)ds = I\).

Замечание: Для поверхностного интеграла первого рода верна аддитивность, то есть для разбиения поверхности \(S\) на подповерхности, сумма интегралов по подповерхностям равна интегралу по поверхности.

Определение: Пусть \(P(x, y, z), Q(x, y, z), R(x, y, z) \in C(S)\). Если \(\exists \displaystyle \lim_{d\to 0}\displaystyle \sum_{i, j = 1}^{N}\left(P(\mathbf{\varphi}(u_i, v_j))\langle \mathbf{n}((u_i, v_j)), \mathbf{e}_x \rangle + Q(\mathbf{\varphi}(u_i, v_j))\langle \mathbf{n}((u_i, v_j)), \mathbf{e}_y \rangle + Q(\mathbf{\varphi}(u_i, v_j))\langle \mathbf{n}((u_i, v_j)), \mathbf{e}_z \rangle\right)\mu(\Pi_{ij}) = I\), то \(I\) – поверхностный интеграл второго рода от дифференциальной формы \(P(x, y, z)dydz + Q(x, y, z)dxdz + R(x, y, z)dxdy\). (скалярные произведения – это по сути соответствующая координата вектора.)

Обозначается: \(\displaystyle\iint\limits_{S^{\mathbf{n}}} P(x, y, z)dydz + Q(x, y, z)dxdz + R(x, y, z)dxdy\)
