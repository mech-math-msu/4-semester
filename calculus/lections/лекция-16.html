Лекция 16

Формула суммирования Эйлера-Маклорена (многочлены Бернулли)

Пусть \(f(x) \in C^1[1, +\infty) \,\,\,\, \forall k \in \mathbb{N}\)

Запишем: \(\displaystyle\int\limits_k^{k + 1} f(x)dx = \displaystyle\int\limits_k^{k + 1} f(x)d(x - k - \frac12) = f(x)(x - k - \frac1k)|_{k}^{k + 1} - \displaystyle\int\limits_k^{k + 1} f'(x)\cdot(x - k - \frac12)dx=\)

\(= \frac12(f(k + 1) + f(k)) - \displaystyle\int\limits_k^{k + 1} f'(x)(\{x\} - \frac12)dx\), где \(\{x\}\) – дробная часть \(x\).

Тут история развития раздваивается (можно записать ряд Фурье периодической функции \(\{x\} - \frac12\) (мы этого делать не будем)) или суммируем равенства выше от \(1\) до \(n - 1\):

\[\displaystyle \sum_{k = 1}^{n - 1}: \,\,\,\, \displaystyle\int\limits_1^n f(x)dx = \displaystyle \sum_{k = 1}^{n}f(k) - \frac12f(1) - \frac12f(n) - \displaystyle\int\limits_1^n f'(x)\left(\{x\} - \frac12\right)dx \implies\]

(формула суммирования Эйлера_маклорена в простейшем варианте)

\[\displaystyle \sum_{k = 1}^{n}f(k) = \displaystyle\int\limits_1^n f(x)dx + \frac12f(1) + \frac12f(n) + \displaystyle\int\limits_1^n f'(x)\left(\{x\} - \frac12\right)dx\]

\(\boxed{\operatorname{I}}\) Если \(f'(x)\) монотонно стремится к нулю при \(x\to\infty\), то \(\exists \displaystyle\int\limits_{0}^{+\infty} f(x)\left(\{x\} - \frac12\right)dx\) и тогда

\[\displaystyle \sum_{k = 1}^{n}f(k) = \displaystyle\int\limits_1^n f(x)dx = \frac12f(n) + C + o(1), \,\,\,\, n \to \infty\]

Забрали в константу существующий (конечный) несобственный интеграл, \(\frac12f(1)\).

Пример

  1. \(f(x) = \frac1{x}\)

\(\displaystyle \sum_{k = 1}^{n}\frac1k = \ln n + \gamma + o(1), \,\,\,\, n \to \infty\) (\(\gamma\) – постоянная Эйлера)

  1. \(f(x) = \ln(x)\)

Проверим, что подходит \(\frac1x \to 0, \,\,\,\,x \to \infty\)

\(\displaystyle \sum_{k = 1}^{n}\ln(k) = n\cdot \ln(n) - n + \frac12\ln(n) + C + o(1), \,\,\,\, n \to \infty\)

\(n! = C_1\cdot n^n\cdot e^{-n}\cdot \sqrt{n}(1 + o(1)), \,\,\,\, n \to \infty\) (формула Стирлинга)

\(\boxed{\operatorname{II}}\) Рассмотрим другие условия для \(f\).

Введем функции:

\(w_1(x) = \{x\} - \frac12\)

\(w_2(x): \,\,\,\, \forall x \in (k, k + 1) \,\,\,\, w_2'(x) = w_1(x), \,\,\,\, w_2(x) \in C(\mathbb{R}), \,\,\,\, w_2\) – периодическая, \(\displaystyle\int\limits_0^1 w_2(x)dx = 0\).

На \((0, 1) \,\,\,\, w_2(x) = \frac{x^2}2 - \frac{x}2 + \frac1{12}\)

\(f(x) \in C^2[1, +\infty)\)

\(\displaystyle \sum_{k = 1}^{n}f(k) = \displaystyle\int\limits_1^n f(x)dx + \frac12(f(n) + f(1)) + f'(x)\cdot w_2(x)|_1^n - \displaystyle\int\limits_1^n f''(x)\cdot w_2(x)dx=\)

\(= \displaystyle\int\limits_1^n f(x)dx + \frac12(f(n) + f(1)) + \frac1{12}(f'(n) - f'(1)) - \displaystyle\int\limits_1^n f''(x)w_2(x)dx\)

Если \(f''(x)\) монотонно к нулю при \(x\to \infty\). Тогда \(\exists \displaystyle\int\limits_0^{\infty} f(x)dx\) и

\[\displaystyle \sum_{k = 1}^{n} = \displaystyle\int\limits_1^n f(x)dx + \frac12f(n) + \frac1{12}f'(n) + C + o(1), \,\,\,\, n \to \infty\]

Пример

\(f(x) = x\cdot \ln(x), \,\,\,\, f''(x) = \frac1{x}\)

\(\displaystyle \sum_{k = 1}^{n}k\cdot \ln(k) =\frac{n^2}2\cdot \ln(n) - \frac{n^2}4 + \frac12n\cdot \ln(n) + \frac1{12}\cdot \ln(n) + C + o(1), \,\,\,\, n \to +\infty\)

\(\displaystyle\int\limits_1^n x\ln(x)dx = \frac{x^2}2\ln(x)|_{1}^n - \displaystyle\int\limits_1^n \frac{x}2d \left(\frac{x^2}2\ln(x) - \frac{x^2}4\right)|_1^n = \frac{n^2}2\cdot \ln(n) - \frac{n^2}4 + \frac14\)

\(\displaystyle \prod_{k =1}^{n}k^k = c_1\cdot n^{\frac{n^2}2}\cdot e^{-\frac{n^2}4}\cdot n^{\frac{n}2}\cdot n^{\frac1{12}}(1 + o(1))\)

Формула суммирования в «общем» виде

Пусть \(g(x) \in C^m[1, +\infty)\) и \(w_m(x)\) последовательные первообразные, которые периодические (период \(1\)), интегрируемые и с нулевым средним, то есть интеграл от \(0\) до \(1\) ноль. (на отрезке \([0, 1]\) это всегда многочлен) (вообще, это сплайн). Тогда

\[\displaystyle \sum_{k = 1}^{n}f(k) = \displaystyle\int\limits_1^n f(x)dx + \frac12(f(n) + f(1)) +\] \[+\displaystyle \sum_{l = 1}^{m - 1}(-1)^{l + 1}(f^{(l)}(n) - f^{(l)}(1))w_{l + 1}(0) + (-1)^{m + 1} \displaystyle\int\limits_1^n f^{(m)}(x)\cdot w_m(x)dx\]

Производящие функции

\(A(x, t) = \displaystyle \sum_{n = 0}^{\infty}w_n(x)\cdot t^n, \,\,\,\, x \in (0, 1), \,\,\,\, t\in (-2\pi, 2\pi)\) сходится (не доказываем).

\(A'_x(x, t) = \displaystyle \sum_{n = 1}^{\infty}w_{n - 1}t^n = t\cdot A\)

Имеем \(A(x, t) = C(t)\cdot e^{xt}\)

Заметим, что \(\displaystyle\int\limits_0^1 A(x, t)dx = 1\).

Проинтегрируем \(A(x, t) = C(t)\cdot e^{xt}\), чтобы найти константу:

\(1 = \displaystyle\int\limits_0^1 C(t) e^{xt} dx = \frac{C(t)}{t}(e^t - 1) \implies\)

\(C(t) = \frac{t}{e^t - 1}\)

\(A(x, t) = \frac{t\cdot e^{xt}}{e^t - 1} = \displaystyle \sum_{n = 0}^{\infty}\frac{B_n(x)}{n!}t^n\)

Определение: \(B_n(x)\) – многочлены Бернулли, а \(w_n = B_n(0)\).

\(A(x, t)\) – производящая функция.

Что-то чего не будет на экзамене

\(\displaystyle\int\limits_0^1 w_{m+1}(x)\cdot w_m(x)dx = \displaystyle\int\limits_0^1 w_{m+1}(x)\cdot w'_{m+1}(x)dx = \frac{w^2_{m + 1}(x)}2|_{0}^{1} = 0\)

То есть многочлены ортогональны, то есть они базис в пространстве.
