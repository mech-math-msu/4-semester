Лекция 15

Свертка функций

Определение: Пусть \(\forall [a, b] \subset \mathbb{R} \,\,\,\, \varphi, \psi \in \mathcal{R}[a, b]\).

(Если \(\exists\)) \(f(x) := \displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t)\psi(x - t)dt\), то \(f(x)\) – свертка функций \(\varphi\) и \(\psi\), которая обозначается \(\varphi\star \psi\).

На \(\varphi\) и \(\psi\) везде дальше в утверждениях добавляются необходимые условия (их надо самим додумывать).

Утверждение: \(\varphi\star \psi = \psi \star \varphi\)

Доказательство:

\[\varphi\star \psi = \displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t)\psi(x - t)dt = (x - t = \tau) =\]

\[=\displaystyle\int\limits_{-\infty}^{+\infty} \varphi(x - \tau)\psi(\tau)d\tau = \psi \star \varphi \,\,\,\,\blacksquare\]

Утверждение: \((\varphi\star \psi)\star \chi = \varphi\star (\psi\star \chi)\)

Доказательство: перестановка пределов \(\,\,\,\,\blacksquare\)

Замечание: (положить ручки сказал Подольский) \(\Delta_s(f(x)) = f(x+s)\) – оператор сдвига. (Это некое общее описание любых разумных процессов от времени)

\(\Delta_s\Delta_t = \Delta_t\Delta_s = f(x + s + t)\) (то есть имеем полугруппу действий)

\[e^{s\cdot \frac{d}{dx}}(f(x)) = \displaystyle \sum_{n = 0}^{\infty}\frac{s^n}{n!} \left(\frac{d}{dx}\right)^n(f(x)) = \displaystyle \sum_{n = 0}^{\infty}\frac{f^{(n)}(x)}{n!}\cdot s^n = f(x + s) = \Delta_s f(x)\]

поэтому все операторы сдвига – экспоненты.

\(e^{At} = u(t), \,\,\,\, \frac{d}{dt}u(t) = tA\)

коммутируют операторы, которые являются функциями друг от друга.

\(-i \frac{d}{dx}(e^{i \lambda x}) = \lambda \cdot e^{i \lambda x}\) (то есть в разумном пространстве – это самосопряженный оператор) и преобразование Фурье – это разложение по собственным числам (спектрам) этого оператора.

конец замечания

Что будет с преобразованием Фурье от свертки?

Утверждение: Для существования светрки достаточно существование интеграла \(\displaystyle\int\limits_{-\infty}^{+\infty}|\varphi(x)|dx, \,\,\,\, |\psi(x)| \leqslant M\).

Доказательство: \(\,\,\,\,\blacksquare\)

Теорема: Пусть \(\exists \displaystyle\int\limits_{-\infty}^{+\infty} |\varphi(x)|dx\) и \(\exists \displaystyle\int\limits_{-\infty}^{+\infty}|\psi(x)|dx\). Тогда \(\frac1{\sqrt{2\pi}}\widehat{\varphi\star \psi}(\lambda) = \widehat{\varphi}(x)\cdot \widehat{\psi}(x)\), где \(\hat{.}\) – преобразование Фурье.

По дороге будем менять порядки интегрирования, вообще \(\exists \displaystyle\int\limits_{-\infty}^{+\infty} |\varphi(x)|dx\) и \(\exists \displaystyle\int\limits_{-\infty}^{+\infty}|\psi(x)|dx\) этого достаточно, но у нас в теореме требовалась еще непрерывность.

Доказательство:

\[\frac1{\sqrt{2\pi}}\widehat{\varphi\star \psi}(\lambda) = \frac1{2\pi} \displaystyle\int\limits_{-\infty}^{+\infty} e^{-i \lambda x} \left(\displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t)\cdot \psi(x - t) dt\right) dx=\]

Меняем порядок интегрирования:

\[= \frac1{2\pi} \displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t) \left(\displaystyle\int\limits_{-\infty}^{+\infty} e^{-i \lambda x} \psi(x - t) dx\right) dt=\]

Заменяем \(x - t = u\):

\[=\frac1{2\pi} \displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t) \left(\displaystyle\int\limits_{-\infty}^{+\infty} e^{-i \lambda (u + t)} \psi(u) du\right) dt =\frac1{2\pi} \displaystyle\int\limits_{-\infty}^{+\infty} \varphi(t) e^{-i \lambda t} \cdot \left(\displaystyle\int\limits_{-\infty}^{+\infty} \psi(u) e^{-i \lambda u}du\right) dt = \widehat{\varphi}(x)\cdot \widehat{\psi}(x) \,\,\,\,\blacksquare\]

Приложения преобразования Фурье

Теорема: (формула суммирования Пуассона) Пусть \(f(x) \in C^2(\mathbb{R}), \,\,\,\, |f(x)| \leqslant \frac{c}{1 + x^2}, \,\,\,\, |f'(x)| \leqslant \frac{c}{1 + x^2}\). Тогда

\[\displaystyle \sum_{k = -\infty}^{+\infty}f(2\pi k) = \frac1{\sqrt{2\pi}}\displaystyle \sum_{n = -\infty}^{+\infty}\widehat{f}(n)\]

(ряд абсолютно сходится в силу условия теоремы)

Доказательство:

\(\varphi(x) = \displaystyle \sum_{k = -\infty}^{+\infty}f(x + 2\pi k)\) эта функция \(2\pi\)-периодическая.

Функция периодическая, значит раскладываем в ряд Фурье:

\[\varphi(x) = \frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx) + b_n(\sin(nx))=\]

Подставляем выражения \(a_n\) и \(b_n\) через интегралы и приходим к ядру Дирихле (было когда-то):

\[= \frac1{2\pi} \displaystyle\int\limits_{-\pi}^{\pi} \varphi(t)dt + \displaystyle \sum_{n = 1}^{\infty}\frac1{\pi} \displaystyle\int\limits_{-\pi}^{\pi} \varphi(t)\cdot \cos(n(x - t))dt\]

\[= \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} \varphi(t)dt + \frac1{\pi} \displaystyle \sum_{n = 1}^{\infty}\displaystyle\int\limits_{-\pi}^{\pi} \left(\displaystyle \sum_{k = -\infty}^{+\infty}f(t + 2\pi k)\right)\cdot \cos(n(x - t)) dt=\]

\[=\frac1{2\pi} \displaystyle\int\limits_{-\pi}^{\pi} \varphi(t)dt + \frac1{2\pi} \displaystyle \sum_{n = 1}^{\infty}\displaystyle\int\limits_{-\pi}^{\pi} \varphi(t)\cdot \left(e^{i n (x - t)} + e^{-i n(x- t)}\right)dt=\]

\[=\frac1{2\pi} \displaystyle \sum_{n = -\infty}^{+\infty}\displaystyle\int\limits_{-\pi}^{\pi} \varphi(t) e^{i n (x - t)} dt = \frac1{2\pi} \displaystyle \sum_{n = -\infty}^{+\infty}\displaystyle\int\limits_{-\pi}^{\pi} \left(\displaystyle \sum_{n = -\infty}^{+\infty}f(t + 2\pi k)\right) e^{i n (x - t)} dt=\]

\[=\frac1{2\pi} \displaystyle \sum_{n = -\infty}^{+\infty}\left(\displaystyle \sum_{n = -\infty}^{+\infty}\displaystyle\int\limits_{-\pi + 2\pi k}^{\pi + 2\pi k}f(\tau)e^{in (x - \tau)}d\tau\right)=\]

\[= \frac1{2\pi} \displaystyle \sum_{n = -\infty}^{+\infty} \displaystyle\int\limits_{-\infty}^{+\infty} f(\tau)e^{in (x - \tau)}d\tau = \frac1{\sqrt{2\pi}} \displaystyle \sum_{n = -\infty}^{+\infty}e^{inx} \hat{f}(n) \,\,\,\,\blacksquare\]

Определение: \(\displaystyle \sum_{n = 0}^{\infty}e^{-n x^2}\) – \(\theta\)-функция (тета-функция).
