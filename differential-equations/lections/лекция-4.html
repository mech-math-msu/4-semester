Лекция 4

Краевые задачи (продолжение)

Теорема: Решение задачи имеет вид \(y(x) = \displaystyle\int\limits_a^b G(x, s)f(x)ds\), где \(G(x, s) = \begin{cases} a(x)y_1(x), a \leqslant x \leqslant s\\ b(s)y_2(x), s \leqslant x \leqslant b \end{cases}\), \(y_1(x), y_2(x)\) – линейно независимые решения \((45)\), удовлетворяющие \((50)\) и \((51)\) соответственно.

  1. \(a(s)y_1(s) = b(s)y_2(s)\)
  2. \(b(s)y_2'(s)-a(s)y_1'(s) = \frac{1}{a_0(s)}\)

Доказательство:

Подставим \(G(x, s)\) в уравнение. \(y(x) = \displaystyle\int\limits_x^b a(s)y_1(x)f(s)ds + \displaystyle\int\limits_a^x b(s)y_2(x)f(s)ds\). Ищем производную. Напоминание (формула Лейбница): \(\frac{d}{dx} \displaystyle\int\limits_a(\alpha)^b(\alpha) F(x, \alpha)dx = \displaystyle\int\limits_a(\alpha)^b(\alpha) F_{\alpha}'(x, \alpha)dx + F(b(\alpha), \alpha)b'(\alpha) - F(a(\alpha), \alpha)a'(\alpha)\).

Получаем: \(y(x) = y_2(x)\displaystyle\int\limits_a^x b(s)f(s)ds + y_1(x)\displaystyle\int\limits_x^b a(s)f(s)ds\)

\(y'(x) = y_2'(x)\displaystyle\int\limits_a^x b(s)f(s)ds + y_2(x)b(x)f(x) + y_1'(x)\displaystyle\int\limits_x^b a(s)f(s)ds- y_1(x)a(x)f(x)\)

Определение: \(Ly = a_0(x)y'' + a_1(x)y' + a_2(x)y= \lambda y (54)\)

\[\begin{cases} \alpha y'(a) + \beta y(a) = 0, \,\,\,\, |\alpha| + |\beta|\ne 0(50)\\ \gamma y'(b) + \delta y(b) = 0, \,\,\,\, |\gamma| + |\delta|\ne 0(51) \end{cases}\]
