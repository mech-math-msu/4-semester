Лекция 3

Краевые задачи: продолжение

Замечание: Далее для линейного уравнения \(n\)-го порядка: \(Ly = a_0(x)y^{(n)} + \ldots + a_n(x)y = f(x), \,\,\,\, f, a_i \in C(\alpha, \beta)\) задано \(n\) краевых условий в различных точках отрезка.

Определение: Если в краевых условиях связаны значения условия в краейвых точках, то задача нелокальная.

Если условия заданы в нескольких точках, то задача многоточечная. (?)

Определение: Если \(f(x)\) и правые части краевых уравнений равны нулю, то соответствующая краевая задача – однородная. (Да, изначально нам дали определение однородной краевой задачи.)

Теорема: (об альтернативе) Пусть задано уравнение \(Ly = a_0(x)y^{(n)} + \ldots + a_n(x)y = f(x), \,\,\,\, f, a_i \in C(\alpha, \beta), \,\,\,\, n \geqslant 2\) с \(n\) краевыми условиями. Тогда

  1. Либо эта задача имеет \(1\) решение при любой непрерывной правой части,
  2. Либо однородная краевая задача при некоторых \(f\) имеет бесконечное множество решений, а при других не имеет решений.

Доказательство:

  1. Пусть \(y_1, \ldots, y_n\) линейно независимые решения \(Ly = 0\). Общее решение – это их линейная комбинация, то есть \(y = C_1y_1 + \ldots + C_n y_n + \tilde{y}(x)\), где \(\tilde{y}(x)\) – частное решение неоднородного.

  2. Подставим \(n\) краевых условий и, перенося \(\tilde{y}(x)\) в правую часть, получим линейную неоднороднуб систему.

  3. Если \(\det \ne 0\), то система имеет \(1\) решение \(\implies\) определены \(c_1 = c_1^0, \ldots c_n = c_n^0\)

  4. Если \(\det = 0\), то однородная система имеет бесконечное число решений.

Пример: Найти наименьшее \(b > 0\), такое, что уравнение \(y'' + b^2y = 0, \,\,\,\, y(0) = 5, \,\,\,\, y(1) = -5\) не имеет решений.

Сначала решаем однородную систему \(\begin{cases} y'' + b^2y = 0\\ y(0) = 0\\ y(1) = 0 \end{cases}\):

Есть линейно независимые решения \(\sin(bx)\) и \(\cos(bx)\). Запишем общее решение: \(y = с_1 \cos(bx) + c_2\sin(bx)\).

\(y(0) = 0 = c_1, \,\,\,\, y(1) = 0 = c_2 \sin(bx) \implies bx = \pi k, \,\,\,\, k \in \mathbb{Z}\)

Определение: Функция Грина – это \(G: \{(x, \xi)|\,\,\,\, x \in [a, b], \,\,\,\, \xi \in [a, b]\}\), такая что

  1. \(\forall \xi \in [a, b] \,\,\,\, y(x) = G(x, \xi), \,\,\,\, x \ne \xi\) удовлетворяет однородному уравнению \(LG(x, \xi) = 0\).
  2. \(G(x, \xi)_{|x = a}\) и \(G(x, \xi)_{|x = b}\) удовлетворяют краевым условиям из определения
  3. \(G(x, \xi)\) непрерывна и \(\forall \xi \,\,\,\, G'_x(x, \xi)\) имеет в точке \(x = \xi\) скачок \(\frac1{a_0(x)}\)

Теорема: Если \(G(x, \xi)\) – функция Грина краевой задачи, то решение этой задачи \(\displaystyle\int\limits_a^b G(x, \xi)f(\xi)d\xi\)
