Лекция 1. Линейные уравнения \(2\)-го порядка

Определение: \(a_0(x)y '' + a_1(x)y ' + a_2(x)y = f(x), \,\,\,\, x \in (\alpha, \beta), \,\,\,\, a_i, f \in C(\alpha, \beta), \,\,\,\, a_0(x) \ne 0\) – линейное диффернциальное уравнение \(2\)-го порядка.

Определение: \(a_0(x)y '' + a_1(x)y ' + a_2(x)y = 0, \,\,\,\, x \in (\alpha, \beta), \,\,\,\, a_i, f \in C(\alpha, \beta), \,\,\,\, a_0(x) \ne 0\) – однородное линейное дифференциальное уравнение \(2\)-го порядка.

Если \(x_0 \in (\alpha, \beta)\), то задача Коши \(\begin{cases} y(x_0) = y_0,\\ y '(x_0) = y_1 \end{cases}\) – имеет при выполнении непрерывности единственное решение, продолжаемое на \((\alpha, \beta)\) (по т.Пикара и т. о продолжении)

Приведение уравнения линейного уравнения \(2\)-го порядка к простейшим формам

  1. Замена \(y = \alpha(x)z, \,\,\,\, \alpha(x) \in C^2(\alpha, \beta)\). Выразим производные, подставим и вынесем \(z\) за скобки. Получим: \(a_0(x)\alpha(x)z '' + (2a_0(x)\alpha '(x) + a_1(x)\alpha(x))z ' + (a_0(x)\alpha ''(x) + a_1(x)\alpha'(x)+a_2(x)\alpha(x))z = f(x)\). Выбираем \(\alpha(x): \,\,\,\, 2a_0(x)\alpha'(x) = -a_1(x)\alpha(x)\). Получаем:

\[\displaystyle\int\limits\frac{\alpha'(x)}{\alpha(x)}dx = -\frac12 \displaystyle\int\limits\frac{a_1(x)}{a_0(x)}dx\] \[\Downarrow\] \[\alpha(x) = Ce^{-\frac12 \displaystyle\int\limits\frac{a_1(x)}{a_0(x)}dx}\] \[\Downarrow\] \[z'' + p(x)z = F(x)\]

Такая форма приведенная.

Замечание: можно уравнение привести к виду выше заменой независимой переменной на \(x = y(t), \,\,\,\, y \in C^2(\alpha_1, \beta_1) \implies\ddot{y} + q(t)y = G(t)\)

  1. Пусть \(a_0 \in C^1\). Хотим \(r_1(x)\left(r_0(x)y'\right)' + a_2(x)y = f(x)\). Найдем \(r_1\) и \(r_0\). Продифференцируем: \(r_1(x)\left(r'_0(x)y' + r_0(x)y''\right) + a_2(x)y = f(x)\).

\[r_0(x)y'' + r_0'(x)y' + \frac{a_2(x)}{r_1(x)}y = \frac{f(x)}{r_1(x)}\]

Умножая уравнение на \(\mu(x)\) и выбирая \(\mu(x)\), чтобы 1 \((\mu(x)a_0(x))' = \mu(x)a_1(x) \implies \frac{\mu'(x)}{\mu(x)} = \frac{a_1(x)-a_0'(x)}{a_0(x)} \implies \mu(x) = Ce^{\displaystyle\int\limits\frac{a_1(x)-a_0'(x)}{a_0(x)}dx}\).

Такая форма называется самосопряженная форма. Удобна, если \(a_2(x) = 0\).

Определение: Точка накопления нулей – это \(x_0 \in (\alpha, \beta): \,\,\,\, \forall U_{\varepsilon}(x_0) \,\,\,\, \exists x \in U_{\varepsilon}(x_0):\,\,\,\, y(x) = 0\).

Теорема: Ненулевое решение уравнения с \(f(x) = 0\) не может иметь точек накопления нулей.

Доказательство: (от противного) Пусть существует \(x_0: \,\,\,\, \displaystyle \lim_{n\to \infty}x_n = x_0\) и \(y(x_n) = 0\). Тогда \(y(x_0) = 0\) в силу непрерывности и \(\forall [x_{n - 1}, x_n]\,\,\,\, \exists \xi_n: \,\,\,\,y(\xi_n) = 0\) (по т.Ролля). Имеем \(\displaystyle \lim_{n\to \infty}\xi_n = x_0, \,\,\,\, y'(x_0) = \displaystyle \lim_{n\to \infty}y'(\xi_n) = 0\). Тогда имеем задачу Коши \(\begin{cases} y'' + p(x)y = 0,\\ y(x_0) = 0\\ y'(x_0) = 0 \end{cases}\), которая имеет единственное решение – тождественный ноль. Противоречие.

Следствие: Уравнение при \(f(x) = 0\) может иметь только конечное число нулей на отрезке.

Доказательство:

Определение: Решение уравнения колеблющееся на \([\alpha, \beta]\), если оно имеет на этом отрезке более одного нуля.

Определение: Решение уравнения колеблющееся на полуоси \([x_0, \infty)\), если \(\forall x_1: y(x_1) = 0 \,\,\,\, \exists x_2 > x_1: \,\,\,\, y(x_2) = 0\). 2

Определение: \(y'' + p(x)y = 0, \,\,\,\, p \in C(\alpha, \beta), \,\,\,\, -\infty \leqslant \alpha < \beta \leqslant +\infty\)

Примеры

Уравнение колебаний \(\ddot{x}+ w^2 x = 0\) колеблющееся, так как \(x(t) = C_1\cos(wt) + C_2\sin(wt)\).

Уравнение \(y'' - k^2x = 0\) не является колеблющимся, так как \(y = C_1e^{kx} + C_2e^{-kx}\).

Теорема: Если в уравнении \(p(x)<0\), то все решения не являются колеблющимися.

Доказательство: (от противного) Пусть существует \(y(x)\) – решение уравнения и \(x_1 < x_2: \,\,\,\, y(x_1) = y(x_2) = 0: \,\,\,\, y(x)\ne 0, \,\,\,\, x \in (x_1, x_2)\). Без ограничения общности \(y(x) > 0, \,\,\,\, x \in (x_1, x_2)\). Заметим, что \(y'(x_1) \ne 0\) (иначе \(y \equiv 0\)) значит \(y'(x_1) > 0\).

\(y'' = -p(x)y > 0\) на \([x_1, x_2] \implies y'(x)\) возрастает, а значит, так как \(y(x_1) = 0 \,\,\,\, y(x_2) > 0\).
