Лекция 7

У любой плоской кривой можем построить репер Френе. \(\alpha'(s) = \tilde{k}(s)\), где \(\alpha\) – угол между касательной и …

Перейдем в \(\mathbb{R}^3\)

Определение: \(\gamma\) – регулярная кривая в \(\mathbb{R}^3 \,\,\,\, x = \mathbf{r}(t) \,\,\,\, x = (x^1, x^2, x^3)\).

На \(\gamma\) можно ввести натуральный параметр – длину дуги.

\(v = \dot{\mathbf{r}}(s), \,\,\,\, n = \frac{\mathbf{r}''}{|\mathbf{r}''|}\)

\(b = v\times n\)

\(n\) – вектор главной нормали,

\(b\) – вектор бинормали,

\((v, n, b)\) – репер Френе.

\(\gamma\) – бирегулярна, если \(\mathbf{r}''\).

Формулы Френе для \(\mathbb{R}^n\)

Теорема: (трехмерные формулы Френе) Пусть в точке \(s_0: \,\,\,\, \mathbf{r}''(s_0)\). Тогда в окрестности точки \(s_0\) существует гладкая функция \(\mathbf{\varkappa}(s)\), такая что \(v' = k'n, \,\,\,\, n' = -kv - \varkappa b, \,\,\,\, b' = \varkappa n\)

Доказательство:

Формула \(\frac{v'}k = n\) следует из определения.

\(Q(s) = (v, n, b)\) – ортогональная матрица, так как векторы – это векторное произведение,

\(Q' = (v', n', b') = Q A\), по лемме \(A\) кососимметричная матрица \(3\times 3\),

\(\left(\begin{array}{ccc} 0 & -a_{21} & -a_{31} \\ a_{21} & 0 & -a_{32} \\ a_{31} & a_{32} & 0 \end{array}\right)\)

Из первой формулы знаем первый столбец матрицы и, следовательно матрица имеет вид:

\(\left(\begin{array}{ccc} 0 & -k & -0 \\ k & 0 & -\varkappa \\ 0 & -\varkappa & 0 \end{array}\right)\)

Формулы доказаны, докажем гладкость \(\varkappa\). Для этого заметим, что \(\varkappa = -(n', b)\).

Определение: \(\varkappa\) – кручение кривой \(\gamma\).

Геометрический смысл кручения

При движении вдоль кривой скорость репера Френе определяется формулами Френе.

Утверждение: Пусть \(\gamma\) бирегулярная кривая. Тогда \(\gamma\) – плоская \(\iff \,\,\,\, \varkappa = 0\) (крчение – тождественный ноль)

Доказательство:

\(\boxed{\Longrightarrow} \,\,\,\, b = b_0 = \operatorname{conts} \implies\) по формулам Френе \(\varkappa =0\)

\(\boxed{\Longleftarrow} \,\,\,\, b = b_0 = \operatorname{conts} (v(s), b_0) = 0 \,\,\,\, (\mathbf{r}', b_0) = 0 \implies (\mathbf{r}(s), b_0) = c_0\)

То есть кручение – это скорость вылезания кривой из плоскости.

Теорема о восстановлении кривой по кривизне и кручению

Теорема: (о восстановлении) \(k(s), \,\,\,\, \varkappa(s) \geqslant 0, \,\,\,\, s \in [0, L], \,\,\,\, L\) – длина кривой.

  1. Пусть \(k_0(s)\) и \(\varkappa(s)\) всюду гладкие функции на отрезке \([0, L]\) и \(k_0 > 0\). Тогда существует бирегулярная кривая \(\gamma\), такая что \(k_0\) – кривизна \(\gamma\), \(\varkappa\) – кручение,
  2. Пусть \(\gamma_1\) и \(\gamma_2\) – бирегулярные кривые и \(k_1(s) = k_2(s)\) и \(\varkappa_1(s)= \varkappa_2(s)\). Тогда существует движение плоскости \(f: \mathbb{R}^3 \to \mathbb{R}^3\), такое что \(f(\gamma_1) = \gamma_2\).

Доказательство:

\[\begin{cases} v' = k_0 n, \,\,\,\, v(0) = v_0\\ n' = -k_0 v - x_0 b, \,\,\,\, n(0) = n_0\\ b' = x_0 b, \,\,\,\, b(0) = b_0 \end{cases}\]

\(Q' = QA_0\)

\(A_0 = \left(\begin{array}{ccc} 0 & -k_0 & 0 \\ k_0 & 0 & \varkappa_0 \\ 0 & \varkappa_0 & 0 \end{array}\right)\)

Хотим воспользоваться теоремой сущесвования и единственности решения для этого поставим начальные условия задачи Коши:

\(Q = Q_0\)

По теореме существования и единственности существуют три вектор-функции \(v(s), n(s), b(s)\), удовлетворяющие системе.

\(x = \mathbf{r}(s) = \displaystyle\int\limits_0^s v(s_1)ds_1 +\mathbf{r}_0, \,\,\,\, \mathbf{r}'(s) = v\)

\(\mathbf{r}'' = v' = k_0 n, \,\,\,\, |\mathbf{r}''| = k_0 \implies k_0\) – кривизна \(\gamma\), \(n(s)\) – главная нормаль, \(b\) – бинормаль.

\(\varkappa_0 = -(n', b)\).

Докажем пункт 2:

Сдвинем начальную точку первой кривой в начальную точку второй. У каждой из кривых в начальной точке имеется репер Френе. Обе тройки ортонормированны и правильно ориентированы, поэтому существует единственной ортогональное преобразование (вращение), переводящее один репер во второй. Из условия теоремы и пункта 1. следует, что реперы совпадают для всех \(s\) и в частности совпадают векторы скорости. Теорема доказана.

Определение: \(\begin{cases} k = k(s)\\ \varkappa = \varkappa(s) \end{cases}\) – натуральные уравнения.

Если мы едем по кривой, то репер Френе просто вращается.

Как найти \(v, n, b, k, \varkappa\) для кривой заданной параметрически?

Пусть кривая задана в некоторой параметризации \(x = \mathbf{r}(t)\).

\(s(t) = \displaystyle\int\limits_{t_0}^t |\mathbf{\dot{r}}(t_1)|dt_1\) будет неудобно считать.

Теорема: В натуральной параметризации \(v = \mathbf{r}', \,\,\,\, n = \frac{\mathbf{r}''}{|\mathbf{r}''|}, \,\,\,\, b = v\times n, \,\,\,\, k = |\mathbf{r}''|, \kappa = \frac{\langle \mathbf{r}', \mathbf{r}'', \mathbf{r}''' \rangle}{|\mathbf{r}''|^2}\)

В произвольной параметризации \(v = \frac{\dot{\mathbf{r}}}{|\dot{\mathbf{r}}|}, \,\,\,\, n = \frac{\dot{v}}{|\dot{v}|}, \,\,\,\, b = v\times n\)

\(k = \frac{|\dot{\mathbf{r}}\times \ddot{\mathbf{r}}|}{|\dot{\mathbf{r}}|^2}, \kappa = \frac{\langle \,\,\dot{\mathbf{r}}, \ddot{\mathbf{r}}, \dddot{\mathbf{r}}\,\,\rangle}{|\dot{\mathbf{r}}\times \ddot{\mathbf{r}}|^2}\)

Доказательство:

Когда \(\mathbf{r}'\) – это производная \(\mathbf{r}(s)\) по \(s\), а \(\dot{\mathbf{r}}\) – это по \(t\), то есть \(\mathbf{r}(s(t))\).

\(\dot{\mathbf{r}} = \mathbf{r}'\cdot \frac{ds}{dt} = \mathbf{r}'\cdot |\dot{\mathbf{r}}|\).

\(\ddot{\mathbf{r}} = \mathbf{r}''\cdot \left(\frac{ds}{dt}\right)^2 + \mathbf{r}'\cdot \frac{d^2s}{dt^2}\).
